
# AI Tool Selection Assistant

This Streamlit app helps users select AI tools based on business function, tool type, and business activity. It uses embedded data from the AI Tool Matrix Excel file.

## Features
- Select business function
- Choose tool type
- Choose business activity
- View and score AI tools
- Get top recommendations with links

## Setup
```bash
pip install -r requirements.txt
streamlit run app.py
```
